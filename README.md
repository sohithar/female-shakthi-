## 🌟 About the Project

**Shakti App** is a next-generation women's empowerment ecosystem designed to address the fragmented and inefficient support systems available to girls and women today. The platform brings together emergency safety tools, scholarship discovery, legal guidance, career opportunities, and community building — all under one unified, intelligent interface.

> *"When women are empowered, families thrive, communities prosper, and nations grow stronger."*

This project was developed as part of the **Smart India Hackathon (SIH)** submission.

---

## ❗ Problem Statement

Today's support ecosystem for girls and women is neither efficient nor unified:

| Problem | Our Solution |
|--------|--------------|
| Overwhelming search process & wasted time | Auto-match and quick-filter based on user profile |
| No unified platform for all empowerment needs | Single app for SOS, scholarships, schemes & opportunities |
| Poor data on resource gaps for government planning | AI-based resource tracking & analytics |
| Existing platforms are outdated | Continuously updated with AI-powered features |
| Unverified resource information causes confusion | Verified Resource ID system for secure data |

## ✨ Key Features

### 🆘 SOS & Safety
- One-touch emergency alert system
- Real-time location sharing with trusted contacts
- Direct connection to emergency services and support centers
- Discreet activation — safe to use in dangerous situations

### 🎓 Scholarships & Schemes
- Centralized, searchable database of government schemes
- Personalized AI-powered scheme matcher
- Financial aid and education opportunity finder
- Links to SWAYAM, MUDRA loans, and more

### 💼 Opportunities
- Verified skill training and vocational programs
- Mentorship network connections
- Career-building resources and job listings
- Entrepreneurship grants and startup support

### ⚖️ Legal & Rights
- Simplified legal rights awareness
- Step-by-step legal procedure guides
- Access to legal counsel and helplines
- Know Your Rights — plain language summaries

### 🤝 Community & Support
- Verified women's groups and local networks
- Safe space for sharing experiences
- Mentorship and peer support matching
- Local initiative participation

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Edge)
- Git installed on your system
- Node.js (for local development server)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/sohithar/female-shakthi-.git
   ```

2. **Navigate to the project folder**
   ```bash
   cd female-shakthi-
   ```

3. **Open in browser**
   ```bash
   # Simply open index.html in your browser
   # OR use a live server extension in VS Code
   ```

4. **Install as PWA**
   - Open the app in Chrome
   - Click the install icon in the address bar
   - The app will be available offline!
